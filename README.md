# HR Workflow Designer

A modern HR Workflow Designer built with React, TypeScript, and React Flow. This application provides a drag-and-drop interface for designing and simulating HR workflows.

## Features

- **Drag-and-Drop Workflow Canvas**: Intuitive interface for building workflow diagrams
- **Configurable Node Forms**: Customize node properties and behaviors
- **Mock API Integration**: Simulate API interactions without backend dependencies
- **Animated Workflow Simulation**: Watch your workflows come to life with smooth animations

## Tech Stack

- **Frontend Framework**: React with TypeScript
- **Workflow Management**: React Flow
- **Styling**: CSS
- **Build Tool**: Vite (implied by npm run dev)

## Installation

Install the project dependencies:

```bash
npm install
```

## Getting Started

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port specified in your console).
